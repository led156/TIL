# 가상 호스트
- 물리적으로는 서버가 1대지만 가상으로 여러 대가 있는 것처럼 설정
- 도메인명은 DNS에 의해서 IP 주소로 변환되어 액세스 되는데
  + 이때 구분할 수 있는 별도 식별자가 존재하지 않을 경우, 가상 호스트를 사용했을 때 어느 쪽에 대한 액세스인지 확인할 수 없게 됨
  + → 호스트명과 도메인명을 완전히 포함한 URI 지정 / Host 헤더 필드에 저장하는 방식으로 구분해야 함.
