- 값을 기록하고, 그 기록한 값을 참조하는 것 (중복 제거)
```
memo = [-1, -1, -1, ... ]

function fibbo(n)
    if (1) != -1
        return (2)
    if n <= 2
        (3) = 1
    else
        (4) = fibbo(n - 1) + fibbo(n - 2)

    return (5)
```


## Tabulation - for문 (바텀업)
```
set dp = [0, 0, 0, ...]

dp[1] = 1
dp[2] = 1

for i = 3 ... i <= n:
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])
```
